## Todo-List

This is a fullstack application that allows managing TODO tasks: creating, removing, and updating status.

### Tech stack

#### Backend

* Node.js
* Express
* Mongoose
* Celebrate

#### Frontend

* Next.js
* React.js
* Formik
* TanStack React Query
* Zustand
* Yup

### Getting Started

```
# Run dev backend
npm run dev:backend

# Run dev frontend
npm run dev:frontend
```
